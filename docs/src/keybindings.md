# Keyboard Shortcuts

Every action in Schemify is accessible via keyboard. Shortcuts are grouped by category.

## File

| Shortcut | Action |
|---|---|
| Ctrl+N | New file |
| Ctrl+O | Open file |
| Ctrl+S | Save |
| Ctrl+Shift+S | Save as |
| Ctrl+T | New tab |
| Ctrl+W | Close tab |

## Edit

| Shortcut | Action |
|---|---|
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+X | Cut |
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Ctrl+A | Select all |
| Ctrl+Shift+A | Deselect all |
| Ctrl+D | Duplicate |
| Delete / Backspace | Delete selected |

## Tools

| Shortcut | Tool |
|---|---|
| Esc | Select |
| W | Wire |
| M | Move |
| L | Line |
| A | Arc |
| C | Circle |
| P | Polygon |
| T | Text |

## Transform

| Shortcut | Action |
|---|---|
| R | Rotate clockwise |
| Shift+R | Rotate counter-clockwise |
| X | Flip horizontal |
| Shift+X | Flip vertical |
| D | Duplicate |
| Arrow keys | Nudge (up/down/left/right) |

## View

| Shortcut | Action |
|---|---|
| Ctrl+= | Zoom in |
| Ctrl+- | Zoom out |
| Ctrl+0 | Reset zoom |
| F | Zoom to fit |
| Z | Zoom to fit (alternate) |
| G | Toggle grid |
| S | Schematic view |
| Shift+V | Symbol view |

## Dialogs & Simulation

| Shortcut | Action |
|---|---|
| Q | Properties dialog |
| Ctrl+F | Find |
| F5 | Run simulation |
| F6 | Refresh plugins |
| : (Shift+;) | Command mode |
