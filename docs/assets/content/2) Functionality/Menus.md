# Menus

Every menu item available in the Schemify toolbar.

---

## File

| Item | Shortcut | Description |
| --- | --- | --- |
| New Schematic | Ctrl+N | Create a blank schematic |
| New Primitive... | — | Open the new primitive dialog |
| Open... | Ctrl+O | Open a file from disk |
| Reload from Disk | Alt+S | Reload the current file |
| Save | Ctrl+S | Save the current file |
| Save As... | — | Save to a new path |
| Save All | — | Save all open documents |
| Export PDF | — | Export schematic as PDF |
| Export PNG | — | Export schematic as PNG |
| Export SVG | — | Export schematic as SVG |
| Export Netlist | — | Export SPICE netlist |
| Print... | — | Print the schematic |
| New Tab | Ctrl+T | Open a new empty tab |
| Close Tab | Ctrl+W | Close the active tab |
| Reopen Closed Tab | Ctrl+Shift+T | Reopen last closed tab |
| (Recent files) | — | Quick-open recently used files |
